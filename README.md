# enursury
Technologies/Tools Used: Bootstrap (HTML/CSS),  Java, MySql, Springboot, Spring tool suite.
Description: I created a nursery web application using Springboot. The main goal of this project is to develop a series of backend APIs so that the user can access the functionalities of the various types of functions one can get while shopping from any online website.
Outcomes:
•	Home Page: Contains various latest products in their category.
•	Login/Register Module: After login, only the user can buy products 
•	Admin Module: The admin can add categories, products, admin, and info about registered users and order updates.
•	User Module: The User can add to their cart his products according to his categories, and fill in all the details required for delivery of the product at the last checkout page.
